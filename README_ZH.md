# TerrariaReinforce
把《Terraria》里的哥布林工匠带到Minecraft里！  
*仍处于开发阶段*

# 命令
`/trf` - 对手上的物品进行锻造. (trf.use)  
`/trf reload` - 重载配置. (trf.reload)  
`/trf clear` - 清理掉手上物品的修饰语. (trf.clear)    
`/trf getcard` - 获取折扣卡. (trf.getcard)
`/trf modifiers` - 查看修饰语列表. (trf.modifiers)
